# Changelog

All notable changes to `pyepubcheck` are tracked here.

## 0.1.0

Initial alpha release candidate.

- Flat package layout under `./pyepubcheck`.
- CLI entry point: `pyepubcheck`.
- Public API entry point: `pyepubcheck.api.validate_path`.
- Console, JSON, XML, and XMP report renderers.
- Behavior-driven coverage imported under `specs/behavior/features`.
- SpecMason configuration for mapping tests to Gherkin scenarios.
- Markdown documentation configured for Sphinx and MyST Parser.
